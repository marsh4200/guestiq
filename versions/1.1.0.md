# GuestIQ 1.1.0 — Home Assistant Automation

- New **Automation** tab: connect GuestIQ to Home Assistant (URL + webhook ID).
- Check-in switches the room's automations on in HA (geyser via AR Smart Load
  Manager, and anything else keyed off the room's occupancy); check-out
  switches them back.
- Room mapping options: prefix + room number (default "Room 3") or use the
  room's name directly.
- Periodic full sync (default 15 min) self-heals missed events; "Sync all
  rooms now" and "Test connection" buttons included.
- Fire-and-forget with retries — HA being offline never blocks reception.
