# GuestIQ v1.0.0

Initial release.

- Guest self check-in via arrival QR (name, contact, ID, address, vehicle, pax)
- Guest records saved & matched for faster returning check-ins
- Reception console: assign room + checkout date, manual check-in, check-out
- Per-room QR codes -> in-room info page (Wi-Fi SSID/password, restaurant, online menu, reception & emergency contacts)
- Room management with occupancy status
- Hotel settings (branding, dining, contacts, public URL for QR links)
- Admin password auth
- One-line installer, host-side update watcher, in-app "Update now"
- Runs on port 9921, SQLite persisted to ./data
